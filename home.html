Go Draw!

Description:

This is a drawing app that uses a webcam and Tensorflow to track a user's hand positions.


It uses Doodlenet to predict what the user has drawn, and gives points for the accuracy of the drawing.

Drawing can be paused by signing the letter 's' in sign language.

How to:

Draw with your hands! (Or mouse)


Using the camera, use your right hand to draw to the screen. Adjust color, width, and tool from the toolbar. The fill tool automatically fills the canvas with whatever color is currently selected.

You can clear or export the image from the toolbar. You can also save and load JSON images to and from the canvas.
